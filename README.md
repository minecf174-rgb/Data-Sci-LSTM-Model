# This is For DataSci Project
To run this project you need

```pip install -r req.txt```

## Install Python venv

**Window**

```py -3.11 -m venv venv```

**Mac/Linux**

```python3.11 -m venv venv```

**Run venv Window**

```venv/Scripts/activate```

**Run venv Mac/Linux**

```source venv/bin/activate```
