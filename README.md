# This is For DataSci Project
To run this project you need

```pip install -r req.txt```

## Install Python venv

**Window**

```bash
py -3.11 -m venv ven
```

**Mac/Linux**

```bash
python3.11 -m venv venv
```

**Run venv Window**

```bash
venv/Scripts/activate
```

**Run venv Mac/Linux**

```bash
source venv/bin/activate
```
